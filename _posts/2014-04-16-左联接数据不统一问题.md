在使用左联接时，遇到问题：<br/>
##### 1、使用tab1,左联接tab2;
##### 2、tab1有4条数据，tab2中对应tab1的4条数据 有>4条数据与之对应；
##### 3、简单使用左联接之后，会出现，数据总数是后面的tab2与之对应的条数。而不是tab1的总数。解决方案
```
select * from tab1 left join
(
  select *  from 
  (
     select a.*,row_number() over(partition by b3 order by b1) r 
     from tab2 a
  )
  where r = 1
) tab12 on a1=b3 
```
其中a1,b3是两表的连接条件中的字段<br/>

)
) GROUP BY QYJG
